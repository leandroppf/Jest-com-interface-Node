# Jest-com-interface-Node

Esse repositório foi criado a fim de utilizar o [Jest](https://github.com/facebook/jest)
 para realizar testes automatizados e utilizar uma interface para mostrar os resultados.

Decidi por utilizar o [Jest-html-reporters](https://github.com/Hazyzh/jest-html-reporters) para gerar uma página em HTML com os 
detalhes de todos os testes realizados.
A decisão foi tomada, pois o Jest-html-reporters possui uma interface bonita, com gráficos e dados bem detalhados sobre cada teste.

### Detalhes sobre os testes

Como o objetivo era apenas aprender a utilizar as ferramentas, os testes realizados foram simples. Utilizei testes com funções 
aritiméticas que fazem chamada as rotas para executar as funções da API testando os seus resultados.

Propositalmente foram criados dois testes que não terão valor positivo, para demonstrar como é feito o resultado para testes
que ficam pendentes e que possuem falhas.
